# ESP32 Bluetooth RC Car 🚗

A small RC car project controlled by an ESP32 microcontroller and a Flutter mobile application via Bluetooth.

---

## 📦 Project Structure

- `esp32-firmware/` – ESP32 firmware (C++)
- `flutter-app/` – mobile control app (Flutter/Dart)
- `hardware/` – electrical schematics (KiCad)
- `docs/` – project documentation

---

## 🧠 Technologies

- ESP32 (C++)
- Flutter / Dart
- Bluetooth Low Energy
- KiCad (electronics design)
